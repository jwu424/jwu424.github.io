TwoPointers
Created 2019-12-26| Updated 2019-12-26|Leetcode
Word count: 1.3k|Reading time: 7 min|Post View:

This is a collection of problems of Two pointers.

Question 1: 3Sum

Given an array nums of n integers, are there elements a, b, c in nums such that a + b + c = 0? Find all unique triplets in the array which gives the sum of zero.

Answer:

We first sort the array, loop the first element and use two pointers to find next two elements that sum of all = 0.

One thing we should notice is that the results should not include duplicate items.

Question 2: 3Sum Closest

Given an array nums of n integers and an integer target, find three integers in nums such that the sum is closest to target. Return the sum of the three integers. You may assume that each input would have exactly one solution.

Answer:

There is only one solution, so we don’t need to check the duplication or the situation with same diff.

We can track the diff and to find the closest one. Similar to the above one.

Question 3: 4Sum

Given an array nums of n integers and an integer target, are there elements a, b, c, and d in nums such that a + b + c + d = target? Find all unique quadruplets in the array which gives the sum of target.

Answer:

We can use the same strategy like 3Sum.

Another solution is to first calculate sum for each pair and then combine pair to get final results.

Question 4: Implement strStr()

Return the index of the first occurrence of needle in haystack, or -1 if needle is not part of haystack.

Answer:

Loop the elem in haystack, if the elem is same with first elem in needle, then we start to compare.

Question 5: Substring with Concatenation of All Words

You are given a string, s, and a list of words, words, that are all of the same length. Find all starting indices of substring(s) in s that is a concatenation of each word in words exactly once and without any intervening characters.

Answer:

The first way to solve this question is to use sliding window. We first record all word in words into a hash map.

And then loop element in s, if word in s not in words or the frequency of word in s is larger than words, then we break.

The second solution is about two pointers.
