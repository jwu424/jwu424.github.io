SlidingWindow
Created 2019-12-26| Updated 2019-12-27|Leetcode
Word count: 1.4k|Reading time: 8 min|Post View:

Sliding window is one of the topic in Leetcode.

Here, I will try to solve some questions. Some of them may not from Leetcode.

Question 1: Subarrays with at most K Different Integers

Given an array A of positive integers, call a (contiguous, not necessarily distinct) subarray of A good if the number of different integers in that subarray is at most K.

Return the number.

Answer:

This is a classic question of sliding window. Basically, we can build a hash map to record the letter we have seen before and compare the length of hash map with K. If length <= K, we add the res. Else, we need to move the windowStart right to reduce the length of hash map.

Question 2: Subarrays with K Different Integers

Leetcode 992

Given an array A of positive integers, call a (contiguous, not necessarily distinct) subarray of A good if the number of different integers in that subarray is exactly K.

(For example, [1,2,3,1,2] has 3 different integers: 1, 2, and 3.)

Return the number of good subarrays of A.

Answer:

For this question, it is exactly the above question. Except we need to run two times, with one K and another K - 1. And then subtract the two results.

Question 3: Get Equal Substrings Within Budget

You are given two strings s and t of the same length. You want to change s to t. Changing the i-th character of s to i-th character of t costs |s[i] - t[i]| that is, the absolute difference between the ASCII values of the characters.

You are also given an integer maxCost.

Return the maximum length of a substring of s that can be changed to be the same as the corresponding substring of twith a cost less than or equal to maxCost.

If there is no substring from s that can be changed to its corresponding substring from t, return 0.

Answer:

This is a classic window sliding question. While loop windowEnd, we sum the cost. If the cur_cost > maxCost, we move the windowStart right and decrease the cur_cost until cur_cost <= maxCost.

Question 6: Grumpy Bookstore Owner

Leetcode 1052.

Today, the bookstore owner has a store open for customers.length minutes. Every minute, some number of customers (customers[i]) enter the store, and all those customers leave after the end of that minute.

On some minutes, the bookstore owner is grumpy. If the bookstore owner is grumpy on the i-th minute, grumpy[i] = 1, otherwise grumpy[i] = 0. When the bookstore owner is grumpy, the customers of that minute are not satisfied, otherwise they are satisfied.

The bookstore owner knows a secret technique to keep themselves not grumpy for X minutes straight, but can only use it once.

Return the maximum number of customers that can be satisfied throughout the day.

Answer:

We can first add all customers[i] where grumpy[I] == 0. And then we move windowStart and windowEnd to keep the length of window.

Question 7: Number of Submatrices That Sum to Target

Given a matrix, and a target, return the number of non-empty submatrices that sum to target.

A submatrix x1, y1, x2, y2 is the set of all cells matrix[x][y] with x1 <= x <= x2 and y1 <= y <= y2.

Two submatrices (x1, y1, x2, y2) and (x1', y1', x2', y2') are different if they have some coordinate that is different: for example, if x1 != x1'.

Answer:

We can use the idea of prefix sum.

  • Calculate the prefix sum for each row.
  • Loop windowStart and windowEnd for columns
  • During the above loop, we loop the row to calculate sum of sub-matrix
  • Using a hash map to record the each current sum.
