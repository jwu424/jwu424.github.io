Leetcode 96. Unique Binary Search Trees

Given n, how many structurally unique BST’s (binary search trees) that store values 1 … n?

Answer:

Given seq 1, 2, …, n, we would like to enumerate each number as root.

Ex: let j as root, then 1, 2, …, j-1 will be left part, j+1, …, n will be right part. Then what we need to do is use left part to construct subtree and get the number of unique ways, same for right part. Final, for root j, then number is product of above two numbers.

Let

G(n): the number of unique binary search trees for a n-length sequence.

F(i, n): the number of unique binary search trees for root i.

G(n) = F(1, n) + F(2, n) + … + F(n, n)

G(0) = 1, G(1) = 1

F(i, n) = G(i - 1) * G(n - i)

Leetcode 123. Best Time to Buy and Sell Stock III

Design an algorithm to find the maximum profit. You may complete at most two transactions.

Note: You may not engage in multiple transactions at the same time (i.e., you must sell the stock before you buy again).

Answer:

$Total = sell_2-buy_2+sell_1-buy_1=sell_2-(buy_2-(sell_1-buy_1))$

Maximum total => Minimum $buy_2-(sell_1-buy_1)$ => Maximum $sell_1-buy_1$

Leetcode 376. Wiggle Subsequence

A sequence of numbers is called a wiggle sequence if the differences between successive numbers strictly alternate between positive and negative. The first difference (if one exists) may be either positive or negative. A sequence with fewer than two elements is trivially a wiggle sequence.

For example, [1,7,4,9,2,5] is a wiggle sequence because the differences (6,-3,5,-7,3) are alternately positive and negative. In contrast, [1,4,7,2,5] and [1,7,4,5,5] are not wiggle sequences, the first because its first two differences are positive and the second because its last difference is zero.

Given a sequence of integers, return the length of the longest subsequence that is a wiggle sequence. A subsequence is obtained by deleting some number of elements (eventually, also zero) from the original sequence, leaving the remaining elements in their original order.
