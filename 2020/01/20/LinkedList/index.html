Leetcode143. Reorder List

Given a singly linked list L: L0→L1→…→L**n-1→Ln,
reorder it to: L0→L**n→L1→L**n-1→L2→L**n-2→…

You may not modify the values in the list’s nodes, only nodes itself may be changed.

Answer:

  • Find the middle one.

  • Reverse the second part

  • Reorder list

  • Notice:

    • Even number of nodes

      Ex: 1->2->3->4->5->6, after reverse,

      1->2->3->4<-5<-6 and 4-> None. Thus, we need to set the next node of the last node in first part to None.

    • Odd number:

      Ex: 1->2->3->4->5, after reverse,

      1->2->3<-4<-5 and the middle 3 -> None. Thus, we don’t need to care about the next node of the last node in first part

  2. At each iteration, insertion sort removes one element from the input data, finds the location it belongs within the sorted list, and inserts it there.
  3. It repeats until no input elements remain.
