Leetcode 201. Bitwise AND of Numbers Range

Given a range [m, n] where 0 <= m <= n <= 2147483647, return the bitwise AND of all numbers in this range, inclusive.

Answer:

5: 0101 ==> m: xxx0…

6: 0110

7: 0111 ==> n: xxx1…

For m and n, their beginning bits are the same until a bit that in m is 0 and in n is 1. For the right part, after AND operation, all will be zero. (For example, 5 AND 6). Thus, the result will be xxx0000.

We only need to remove the right 1-bit in n until m == n.

Leetcode 231. Power of Two

Given an integer, write a function to determine if it is a power of two.

Answer:

For the number of power of two, its binary form has only one 1-bit. 2==>0010, 4==>0100, 8==>1000 and so on.

n & n-1= 0 if n is power of two.
